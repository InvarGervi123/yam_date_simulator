ים דייט סימולטור - גרסה מתוקנת

איך להפעיל:
1. חלץ את קובץ ה-ZIP.
2. פתח index.html בדפדפן.

מה תוקן:
- המשחק עובד גם אם חסרות תמונות או אודיו.
- אין קריסה אם קובץ לא נמצא.
- story.js עבר ל-ID של סצנות במקום מספרים, כדי שלא יישבר כשמוסיפים סצנות.
- יש כפתור "לשחק שוב" בכל סוף.
- הטקסט תומך בירידות שורה.
- יש סיפור ארוך, דרמטי, מטומטם, עם הרבה סופים.

קבצים חשובים:
- story.js = כל הסיפור והבחירות.
- game.js = המנוע של המשחק.
- style.css = העיצוב.
- images = תמונות.
- audio = אודיו.

כדי להוסיף תמונה לסצנה ב-story.js:
bg: "images/name.jpg"
character: "images/name.png"

כדי להוסיף מוזיקה:
music: "audio/name.mp3"
